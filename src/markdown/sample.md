## Introduction

Markdown is a lightweight markup language with plain text formatting syntax. It's designed to be easy to read and write, and it's often used for documentation, notes, and online content.

## Headings

You can create headings using the `#` symbol. The number of `#` symbols determines the heading level.

## Text Formatting

- **Bold text** using `**bold text**` or `__bold text__`\* _Italic text_ using `*italic text*` or `_italic text_`\* ~~Strikethrough text~~ using `~~strikethrough text~~`

## Lists

### Unordered Lists

### Ordered Lists

## Links

You can create links using the following syntax: `[Link text](URL)`

For example: [Google](https://www.google.com)

## Images

You can embed images using the following syntax: `![Alt text](image URL)`

For example: `![Markdown Logo](https://markdown-here.com/img/icon256.png)`

## Code

### Inline Code

Use backticks (`) to format inline code:  `print("Hello, world!")`

### Code Blocks

Use triple backticks (```) to create code blocks. You can specify the language for syntax highlighting.

```python 
def greet(name):  """Greets the person passed in as a parameter."""  print(f"Hello, {name}!") greet("World")
```

## Blockquotes

Use the `>` symbol to create blockquotes:

> This is a blockquote.> It's often used to highlight important text.

## Horizontal Rule

You can create a horizontal rule using three or more hyphens (`---`), asterisks (`***`), or underscores (`___`).

##  Escape Characters

If you need to use a Markdown formatting character literally, you can escape it with a backslash (`\`).

For example:  `\*This is not italic\*` will render as *This is not italic*.```
